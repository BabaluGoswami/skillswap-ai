# 🚀 SkillSwap AI

An AI-powered MERN Stack web application that enables students to exchange skills, connect with peers, and collaborate through real-time chat.

SkillSwap AI allows users to showcase the skills they can teach, discover new skills to learn, send skill swap requests, and communicate seamlessly with other students on the platform.

## 🌐 Live Demo

**🔗 Visit the Website**

https://skillswap-ai-khaki.vercel.app/

---

## 📌 About The Project

SkillSwap AI is a full-stack MERN application built to simplify peer-to-peer skill exchange among students. The platform provides an intuitive interface where users can create profiles, manage their skills, send collaboration requests, and chat in real time after a successful match.

The project focuses on authentication, scalable backend architecture, responsive UI, and real-time communication.

---

## ✨ Features

- 🔐 Secure JWT Authentication
- 👤 Student Profile Management
- 🎯 Skill Exchange Requests
- 💬 Real-Time Chat using Socket.IO
- 📊 Dashboard & Platform Statistics
- 📱 Fully Responsive User Interface
- 🛡️ Input Validation using Zod
- ⚡ Fast REST API with Express.js

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/BabaluGoswami/skillswap-ai.git
```

### Install Dependencies

#### Backend

```bash
cd backend
npm install
npm run dev
```

#### Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## 🔑 Environment Variables

### Backend

```env
PORT=5000
MONGODB_URI=
JWT_SECRET=
JWT_EXPIRES_IN=7d
FRONTEND_URL=http://localhost:5173
```

### Frontend

```env
VITE_API_URL=http://localhost:5000
```
